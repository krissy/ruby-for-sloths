Learn To Code using Ruby (and sloths)

Workshop

This workshop is given by software engineer Kriselda Rabino (@krissygoround).

It introduces basic coding concepts using the Ruby programming language and has a few too many sloth analogies in it due to Kris' unhealthy obsession with these slow, amusing creatures.

View Slides

Topics

  1. So, who's this Ruby kid anyway?
  2. Playing with Ruby.
  3. Mistakes are inevitable; accept them, embrace them. (aka Exceptions are your new BFF)
  4. What does 'Object-Oriented' mean? How to think in things and classes.
  5. Ruby's basic data types: numbers and strings and arrays, oh my!
  6. Decisions, decisions. Conditional Statements are here to help.
  7. Doing things over and over (now it's getting a bit loopy)
  8. Your turn! Let's make some code.
  9. Continue your code-ventures.
